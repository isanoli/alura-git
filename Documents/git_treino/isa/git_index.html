O que é o Git?

O Git é um sistema de controle de versões, ou VCS (Version Control System).

Git Básico: Criando um repositório

1. $ cd + nome do local // localizar a pasta

2. $ ls // abre o local

3. $ git init // inicia o Git, cria o repositório

4. $ git status // mostra o status

5. $ git add // add o arquivo

Git Básico: Comitando pela primeira vez

1. $ git status // depois que o arquivo foi adicionado, verificamos o status

2. $ git commit -m "Mensagem descritiva" // comando para comitar alteração (check point)

3. $ git status // para verificar se ainda há algo para ser comitado

Git Básico: Comitando modificações

1. $ git status // para verificar se a alteração já foi registrada

2. $ git add // para selecionar o file que será comitado

3. $ git commit -m "Mensagem descritiva" // comitando alteração

4. $ git status // para verificação

Git Básico: Exibindo histórico

$ git log // para exibir o histórico

$ git log --oneline // para exibir o histórico resumido. Você pode utilizar --graph para a descrição completa dos commits

$ git log --pretty="format: %an: %s %cD" // para exibir o histórico de forma personalizada, confira em devhints.io/git-log-format

$ git log -p // para exibir o histórico detalhado

Git Básico: Configurações de usuário

$ git config --local ou --global // para definir em que extensão você deseja acessar as configs

$ git config --local user.name "Nome que você quer definir" // para definir o nome do usuário

$ git config --locar user.email "example@123.com.br" // para definir o e-mail do usuário

$ git config user.name // exibe o nome do usuário

$ git config user.email // exibe o e-mail do usuário!
